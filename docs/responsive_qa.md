# Responsive & UX QA Kısa Liste

- Tablolar: Tasks vb. geniş tablolarda `overflow-x-auto` ve min-width; mobilde yatay scroll.
- Dropdown/menü: Arka plan opak, dokunmatik hedefler yeterli; ekran genişliğinde sığma.
- Formlar: Küçük ekranda tek sütun; tarih/saat picker’ları taşmıyor.
- Kart grid: md breakpoint’te 1-2 kolon; spacing kontrolü.
- Boş durum/hata: Mesaj + aksiyon butonu; yükleniyor iskeleti.
